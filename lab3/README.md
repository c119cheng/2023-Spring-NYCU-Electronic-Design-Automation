The input data save in ./case[n]/

# To complile
```bash
    make
```

# To run 3 sample case
the output will save in ./case[n]/
```bash
    make run1
    make run2
    make run3
```

# To run with any input file
```bash
    ./Lab3 [input file name] [output file name]
```

# To delete binary and object file
```bash
    make clean
```
